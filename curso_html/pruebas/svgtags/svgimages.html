Agregando svg de una manera rapida a nuestro HTML

Pros

  • Sintaxis similar a la de un img comun ademas de tener disponible un atributo alt
  • Puedes hacer la imagen linkeable a traves del uso del tag <a>

Cons

  • No puedes manipular la imagen con javascript
  • Si quieres controlar el contenido SVG con CSS debe ser a traves de inline-style los links externos a CSS que manipulen esto no tienen efecto sobre el SVG
  • No puedes aplicar estilo con pseudo clases ejemplo :focus

Ejemplo de como incrustar un SVG en linea en nuestro documento HTML

Pros

  • Mejora el rendimiento de carga del documento HTML
  • Se puede manipular con CSS
  • Puedes usar interacciones con pseudo clases :focus
  • Se puede convertir en hipervinculo agregando el tag <a>

Cons

  • Este metodo es efectivo solo si utilizas el SVG en un solo sitio, su duplicacion hace el codigo inmantenible a largo plazo
  • El codigo extra de SVG incrementa el tamaño de tu HTML
  • El navegador no cachea los SVG inline
  • Puedes incluir contenido alternativo con un fallback <foreignObject> pero los navegadores que soportan SVG descargan cualquier imagen de respaldo, asi que se debe tener en consideracion esto a la hora de querer dar soporte a navegadores mas antiguos
